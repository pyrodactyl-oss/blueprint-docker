## Reporting a Vulnerability

- To report a vulnerability in private, feel free to send an email to `loki@crazycoder.dev`
- Note that this policy is strictly for this repository, the container image for Blueprint. To report a vulnerability in the Blueprint project itself, refer to the [Blueprint Security Policy](<https://github.com/BlueprintFramework/framework/security>)
